# Archive

This directory stores historical project artifacts that should not guide current implementation.

- `ui.example.js` is an early dark-theme React sketch. It predates rule-group block pages, handoff pages, and the current light-first design system.
- `i18n-plan-2026-06-27.md` is the original multilingual implementation plan. The current maintenance guide is `project-docs/i18n.md`.

Current sources of truth:

- Product behavior: `prd.md`
- Visual system: `DESIGN.md`
- Runtime architecture: `project-docs/architecture.md`
- Runtime localization: `project-docs/i18n.md`
- Actual UI implementation: `src/popup/`, `src/options/`, `src/block/`, and `src/handoff/`
