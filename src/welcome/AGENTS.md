# Welcome Surface Guidelines

The welcome page is the extension's branded home surface. It explains FocusGate / 守界 before setup, then hands users to `options.html?onboarding=1`.

- Keep this page brand-first, not settings-heavy. Configuration belongs in Options.
- User-facing copy is Chinese. Keep “晚安边界” scoped to the default sleep rule group.
- Link product claims back to `prd.md` and visual decisions back to `DESIGN.md`.
- Do not add remote assets or analytics. This page must work inside the packaged extension and local Vite dev server.
