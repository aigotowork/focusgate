FAQ

Does FocusGate upload my browsing history?

No. The current MVP does not upload browsing history. The extension only checks locally whether domains match rules configured by the user.

Why does it need broad site permissions?

Users can add any HTTP(S) domain to a rule group, so the extension needs to check domains and show reminder overlays on user-selected sites. It does not read page body text, form inputs, account passwords, or page titles.

Can it completely stop me from bypassing limits?

No. FocusGate is designed to create visible boundaries, reminders, and unlock friction. It is not anti-uninstall software, a cross-browser lock, or system-level enforcement.
