FocusGate / 守界

规则组正在守住边界

1创建规则组：晚安、工作、戒断
2设置时间、星期和受限网站
3配置提醒、阻断页和临时解锁
4到点后由浏览器扩展执行边界

适合这些边界场景

守界不是惩罚工具，也不承诺无法绕过。它把你清醒时设下的决定，带到容易分心的时刻。

睡前放下

默认“晚安守护”可以限制视频、社区、新闻和其他睡前高刺激网站。

工作专注

为工作日创建规则组，让娱乐网站和信息流在关键时段变得更难打开。

戒断节制

通过提醒、阻断页和解锁摩擦，把周末或休息日还给更慢的现实生活。

本地优先的隐私设计

当前 MVP 不提供账号系统，不进行云同步，不出售用户数据。规则组、受限域名、临时解锁和统计默认保存在浏览器本地。扩展不会读取页面正文、表单输入、账号密码或页面标题。
