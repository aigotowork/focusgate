常见问题

守界会上传我的浏览记录吗？

当前 MVP 不上传浏览记录。扩展只围绕用户设置的域名规则在本地判断是否需要提醒或阻断。

为什么需要较宽的站点权限？

用户可以把任意 HTTP(S) 域名加入规则组，因此扩展需要在这些用户选择的网站上判断域名并显示提醒浮层。它不会读取页面正文、表单输入、账号密码或页面标题。

可以彻底防止我绕过限制吗？

不可以。守界的目标是建立可见边界、提醒和解锁摩擦，而不是防卸载、防换浏览器或系统级强制管控。
